About

UX ELEment (previously called PSDETCH) is an open source ux tool for developers. It is made by developers like you, who also suffer from the gap between design and development. It cares development experience after designers finishing the design. UXELE provides 4 major features: Prototyping, Inspecting, Reviewing and Collaboration. It will accompany your development till it is accurately done. Start now, no registration is needed.

Prototype

Got a static design? UXELE prototyping creates user flow with simple clicks. What about designs with user flow, like Adobe XD? No problem. UXELE can import any design files. With UXELE presenting, it is also able to play the prototyping on browser, mobile and desktops.

Collabarate

In UXELE, collaboration means two things: 1. user could annotate on the design file; 2. user can share the design. For first point, UXELE has built in annotation tools. Although UXELE is a pure client app, with its tight integration to major public cloud storage services like GDrive, OneDrive and DropBox, UXELE can help users to work on same file without compromising privacy.

Inspect

UXELE can inspect a design. It extracts layers/assets and save them into different format. Vector design is fully supported. Even more, it converts layers into code like CSS / Scss / HTML which can be copied. Besides, UXELE also measures size of elements, their distance and color of pixels. Configurations like screen scale and size unit are also supported.

Review

UXELE provides pixel-to-pixel visual comparison to original designs to help developers to identify the difference.
